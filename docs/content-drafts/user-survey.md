## How to deploy

Pick one of the following platforms to host this survey:

### Option A: Google Forms

1. Go to [forms.google.com](https://forms.google.com) and create a new form.
2. Copy each question above, selecting the matching input type (checkboxes, multiple choice, linear scale, short answer).
3. For question 9 (NPS), use a "Linear scale" from 1 to 10.
4. Under Settings, enable "Collect email addresses" if you want follow-up capability.
5. Share the form link in the README, GitHub Discussions, and release notes.

### Option B: Tally.so

1. Go to [tally.so](https://tally.so) and create a new form (free tier supports unlimited forms).
2. Tally supports markdown-style formatting, so you can paste sections almost verbatim.
3. Use "Checkbox" blocks for multi-select questions and "Multiple choice" for single-select.
4. For the NPS question, use Tally's built-in "NPS" block type.
5. Embed the form directly in the llm-wiki site using an iframe or link from the footer.

### Option C: GitHub Discussion poll

1. Go to the repository's Discussions tab and create a new discussion in the "Polls" category.
2. GitHub polls support single-choice questions only, so split multi-select questions into separate polls or use a pinned discussion with reaction-based voting.
3. This option keeps everything within GitHub but is the most limited in question types.
4. Best suited as a lightweight complement to a full form rather than a standalone survey.
