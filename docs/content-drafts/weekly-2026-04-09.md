## LinkedIn Version (~200 words)

This week llm-wiki hit a milestone I didn't expect this early.

v0.9 shipped with two features I've wanted since day one: **project topics** (tag your projects by domain -- backend, infra, ML, etc.) and **agent labels** -- colored badges for Claude, Codex, Copilot, Cursor, and Gemini so you can tell at a glance which AI assistant handled which session.

But the bigger story is the 85 marketing playbook actions we completed in a single push: 32 new files including a Dockerfile, Nix flake, a GitHub Action for CI, 11 docs and tutorials, 18 content drafts, and 6 integration scaffolds. The project now has first-class packaging for Docker, Nix, and GitHub Actions out of the box.

On the maintenance side: 3 Dependabot PRs merged (actions/cache v5, upload-pages-artifact v4, github-script v8), GitHub Discussions are now live, 20 topic tags on the repo, and 39 new tests covering Jupyter export, webhook handling, and the Alfred workflow.

If you've been meaning to try llm-wiki, this is a good week to start.

Demo: https://pratiyush.github.io/llm-wiki
GitHub: https://github.com/Pratiyush/llm-wiki

Star it, open a Discussion, or just `pip install llm-wiki` and see what your session history looks like as a wiki.
